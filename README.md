# 🎮 坦克大战 (Tank Battle)

经典 FC 坦克大战网页版，使用纯 HTML5 Canvas 和 JavaScript 实现。

## 🎯 游戏特色

- **经典玩法**：复刻经典的 Battle City 坦克大战
- **4个关卡**：难度递增的地图设计
- **多种地形**：砖墙（可摧毁）、钢铁（不可摧毁）、河流（不可通行）、树丛（视觉遮挡）
- **道具系统**：加速⚡、射速🔥、护盾🛡、额外生命❤
- **屏幕震动**：爆炸时的震动反馈
- **粒子效果**：碎片和爆炸粒子特效
- **护盾系统**：玩家出生时短暂无敌

## 🕹️ 操作方式

| 按键 | 功能 |
|------|------|
| `WASD` / `方向键` | 移动坦克 |
| `空格` / `J` / `F` | 发射子弹 |
| `P` | 暂停/继续 |

## 🎮 游戏规则

1. 消灭所有敌方坦克即可通关
2. 保护底部中央的基地（老鹰标志），基地被摧毁则游戏结束
3. 每关有 20 辆敌方坦克，关卡越高敌人越多越快
4. 玩家有 3 条生命，被击中会损失一条命
5. 摧毁砖墙有概率掉落道具

## 🚀 运行方式

直接双击 `index.html` 在浏览器中打开即可，无需任何依赖或服务器。

或者在项目目录下运行：

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .
```

然后访问 `http://localhost:8000`
